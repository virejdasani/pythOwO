<h3 align="center">Variables</h3>
<p align="center">Variables can be decluwuared using the keyword <code>pwease</code>.</p>

```py
pwease baka = 100
pwease tehe = "two"
pwease chan = 25
pwease baka = baka + 1
pwease tehe = 500
pwease chan = chan * 2

pwint(tehe)
pwint(baka)
pwint(chan)
```
